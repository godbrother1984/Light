Test Running Numbers System

การทดสอบระบบ Running Numbers

1. ไปที่ Master Data Manager

2. คลิก tab "รหัสอ้างอิง"

3. กด "เติมข้อมูลเริ่มต้น (Seed Data)" เพื่อเพิ่มข้อมูล Running Numbers

4. ไปที่ สร้างงานใหม่ เพื่อทดสอบการสร้าง Job ID ใหม่

ข้อมูลที่ควรเห็นในระบบ
Running Numbers Types:
  • JOB - รหัสงาน (Prefix: JOB)
  • REPORT - รหัสรายงาน (Prefix: RPT)
รูปแบบ Job ID ใหม่:

JOB-YYYY-MM-XXX เช่น JOB-2568-09-001

ขั้นตอนการทดสอบ:
  1. เข้าไปที่หน้า Master Data Manager
  2. คลิกที่แท็บ "รหัสอ้างอิง"
  3. ถ้ายังไม่มีข้อมูล ให้กด "เติมข้อมูลเริ่มต้น"
  4. จะเห็นตารางแสดง Running Numbers
  5. ไปสร้างงานใหม่ จะได้ Job ID ในรูปแบบใหม่
